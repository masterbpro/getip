# getip
Console application to get information about IP address
### Requirements:

* Python 3.5+
* setuptools ~= 50.3.2
* requests ~= 2.24.0

### Installation

``` bash
$ pip3 install git+https://github.com/masterbpro/getip@master
```

### Get information about host IP address
``` bash
$ getip
```
#### Get information about `24.48.0.1` (for example)
```bash
$ getip 24.48.0.1
```
